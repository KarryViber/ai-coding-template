# CLAUDE.md

> 首次使用？运行 `bash setup.sh` 选择工作模式并配置此文件。

这是一个项目模板。请运行 setup.sh 来初始化。

## 可用工作模式

| 模式 | 用途 | Agent Team |
|------|------|-----------|
| **build** | 从零构建新产品/功能 | researcher + developer + reviewer |
| **extend** | 在现有代码上修改/拓展 | explorer + developer |
| **fix** | 定位问题并修复 | 单 agent |
| **explore** | 调研验证，技术选型 | parallel researchers |
